# JavaScript_Mini_Project
Three beginner-friendly JavaScript mini projects: Arrow Key Movement Game, Key Press Event Viewer, and Background Color Switcher. Built using HTML, CSS, and Vanilla JavaScript.



# 🎯 JavaScript Mini Projects Collection

This repository contains 3 beginner-friendly JavaScript mini projects built using **HTML, CSS, and Vanilla JavaScript**.

These projects focus on DOM manipulation and keyboard/mouse events.

---

## 🚀 Projects Included

### 1️⃣ Arrow Key Movement Game

Move a box using arrow keys.

#### ✨ Features:
- Move box with Arrow Keys
- Emoji changes on key press 😨
- Emoji resets on key release 😊
- Background color changes while moving
- Uses `keydown` and `keyup` events

#### 🧠 Concepts Used:
- `addEventListener`
- Keyboard Events
- `style.top` and `style.left`
- Switch statement
- DOM manipulation

---

### 2️⃣ Key Press Event Viewer

Displays details of the key pressed in real time.

#### ✨ Features:
- Shows:
  - `key`
  - `keyCode`
  - `code`
- Special handling for Space key
- Dynamic table generation

#### 🧠 Concepts Used:
- Keyboard events
- Template literals
- `innerHTML`
- Event object properties

---

### 3️⃣ Background Color Switcher

Change the background color by clicking buttons.

#### ✨ Features:
- Multiple buttons
- Background changes based on button `id`
- Uses event delegation concepts

#### 🧠 Concepts Used:
- `querySelectorAll`
- `forEach`
- Click events
- `event.target`

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 🎯 Learning Goals

- Understand DOM manipulation
- Work with keyboard and mouse events
- Improve JavaScript fundamentals
- Practice event handling
